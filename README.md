# Youtube Thumbnail Grabber

This is a simple app that allows you to grab the thumbnail of a youtube video.

## Run Locally
Open Source, Free to use, No ads, No tracking, No data collection, No privacy issues.

**Prerequisites:**  Node.js Supabase Account

1. Install dependencies:
   `npm install`

2. Set the `SUPABASE_URL` and `SUPABASE_KEY` in [.env.local](.env.local) to your Supabase URL and API key

3. Run the app:
   `npm run dev`

4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
